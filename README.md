# 📊 Interactive Financial Transactions & Fraud Dashboard

An interactive **Streamlit dashboard** for exploring, analyzing, and visualizing financial transaction data, with a focus on identifying trends and potential fraud patterns.

---

## 🚀 Overview

This project provides a simple yet powerful interface to:

* Filter and explore transaction data dynamically
* Visualize key insights through interactive charts
* Perform grouped statistical analysis
* Export filtered data for further use

Built with **Streamlit**, this dashboard is ideal for data analysis, fintech exploration, and portfolio projects.

---

## 🧩 Features

### 🔍 Data Filtering

* Filter by **date range**
* Multi-select filters for **categorical variables**
* Real-time updates across all charts

### 📈 Visualizations

* **Time Series Line Chart** (transaction trends)
* **Histogram** (distribution of values)
* **Bar Chart** (average per category)
* **Pie Chart** (category distribution)
* **Correlation Heatmap** (numerical relationships)

### 📊 Data Analysis

* Group-by aggregation:

  * Mean
  * Sum
  * Count

### 📥 Export

* Download filtered dataset as a **CSV file**
